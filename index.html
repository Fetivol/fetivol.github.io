    AIR app

    Dla jakości powietrza było użyto Air api airly eu ze strony AirlyDeveloper
    Koordynaty są brane z Szerokości oraz DŁugości
    • Przy pomocy navigator.geolocation.getCurrentPosition(success[, error[, options]]) jest pobierana pozycja na mapie
    • Po czym ta pozycja jest dodawana do AIRApi, które zawiera dane jakości powietrza
    • Poniżęj jest przedstawiona część kodu który pomaga w pokazywaniu powietrza

      function(info) {
      $('#air_status').text(info.current.indexes[0].description);
      $('#air_status').parent().css('background-color', info.current.indexes[0].color);}

    Pogoda

    Dla Pogody było użyto API ze strony openweathermap.org
    Koordynaty są pobierane tak samo jak w pliku dla Jakośc powietrza, przy pomocy Navigator.Geolocation
    • W tym api jest możliwość dodania swojej pozycji za pomocą ID_Misata, Geopozycji, ZIP kodu.
    • Ponższy kod jest użyty do otrzymania najbardziej potrzebnej informacji dla naszej aplikacji - pogoda, miasto, temperatura
      var icon_link = "http://openweathermap.org/img/w/";
      $('#weather_info').text(info.list[0].weather[0].description);
      $('#city_name').text(info.city.name);
      $('#weather_temp').text(Math.round(info.list[0].main.temp) + "°C");
    • Tak że w Aplikacji jest wyświetlana pogoda z odstępem o 3 godziny do przodu. Co pomoże wziąć parasolkę ze sobą do pracy
